## Features
* 新增一筆支出資料
* 可以依照「資料類別」篩選同類的支出資料
* 統計加總每一筆支出
* 可以編輯該筆資料
* 可以刪除該筆資料
* 點擊 Expense Tracker 即可回到首頁
